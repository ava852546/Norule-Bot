# NoRule Bot 重點功能與指令教學

本文件是快速索引版，整理「重點功能」、「完整功能清單」與「常用指令教學」。

## 重點功能
- 音樂播放：支援 YouTube 關鍵字/連結與 Spotify 連結轉播。
- 互動式音樂面板：按鈕控制播放、跳過、停止、離開、循環、Autoplay。
- 音樂面板資訊：顯示 `頻道`（標記頻道）與 `列隊`（待播數量/清單）。
- 私人包廂：使用者進入觸發語音後自動建立個人包廂，無人自動刪除。
- 通知模板：成員進出與語音進出可用表單編輯模板並預覽。
- 多類型日誌：訊息、指令使用、頻道事件、身分組、管理事件可獨立頻道。
- 多伺服器設定：每個伺服器使用 `guild-configs/<guildId>.yml` 獨立保存。
- 多語系：內建繁中與英文，可切換。

## 功能清單
- 音樂系統：`/play`、搜尋下拉選單、播放佇列、重複播放、Autoplay、自動離開。
- 管理工具：`/delete-messages`（含二次確認、數量限制、跨頁搜尋）。
- 伺服器通知：成員加入/離開、語音加入/離開/移動通知。
- 訊息與伺服器日誌：
  - 訊息編輯/刪除
  - 指令使用
  - 頻道建立/刪除
  - 身分組變更
  - 封禁/踢出管理事件
- 私人包廂控制：上鎖/解鎖、人數上限、改名（由 `/包廂設定` 操作）。
- 設定中心：`/settings info`、`/settings module`、`/settings logs`、`/settings music`、`/settings template`。

## 快速開始
1. 在 `config.yml` 填入 `token`。
2. 啟動 Bot（建議 jar 方式）：
```bash
java -Dfile.encoding=UTF-8 -jar target/discord-music-bot-1.0.0-all.jar
```
3. 進入 Discord 使用 `/help` 檢查指令是否已註冊。
4. 先設定：
   - `/settings logs`：設定預設日誌頻道
   - `/settings music`：設定音樂指令頻道與私人包廂觸發頻道
5. 開始使用 `/play` 播放音樂。

## 指令教學

### 音樂指令
- `/join`：讓 Bot 進入你的語音頻道。
- `/play query:<關鍵字或URL>`：播放歌曲或加入佇列。
- `/skip`：跳過目前歌曲。
- `/stop`：停止播放並清空佇列。
- `/leave`：讓 Bot 離開語音頻道。
- `/music-panel`：建立新的音樂控制面板。
- `/repeat mode:<off|single|all>`：設定循環模式。

範例：
```text
/play query:YOASOBI 群青
/play query:https://www.youtube.com/watch?v=xxxx
```

### 刪除訊息
- `/delete-messages channel channel:<頻道> amount:<1-99>`
- `/delete-messages user user:<使用者> amount:<1-99>`
- 中文別名：`/刪除訊息`

說明：
- `amount` 不填預設為 `99`。
- 會先顯示確認按鈕，避免誤刪。
- 只能刪除 Discord API 允許範圍（14 天內）的訊息。

### 設定指令
- `/settings info`：查看分頁化設定總覽。
- `/settings reload`：重新讀取目前伺服器設定。
- `/settings reset`：開啟重設選單（有二次確認）。
- `/settings language code:<en|zh-TW>`：切換伺服器語言。
- `/settings template`：用下拉 + 表單編輯通知模板。
- `/settings logs`：用選單設定各類日誌頻道。
- `/settings music`：用選單調整音樂/包廂相關設定。
- `/settings module`：一鍵切換各模組開關。

### 模組開關（`/settings module`）
目前可切換：
- 通知啟用
- 訊息日誌啟用
- 成員加入通知
- 成員離開通知
- 語音日誌
- 指令使用日誌
- 頻道建立/刪除日誌
- 身分組變更日誌
- 封禁/踢出管理日誌
- 音樂自動離開啟用
- 音樂 Autoplay 啟用
- 私人包廂啟用

### 私人包廂
- `/settings music` 設定「私人包廂觸發頻道」。
- 使用者進入觸發語音後，Bot 自動建立「使用者名稱 的包廂」並移動使用者。
- `/private-room-settings` 或 `/包廂設定`：
  - 上鎖/解鎖
  - 設定人數（1-99，留空為無上限）
  - 更改名稱（最多 10 字）
- 設定人數與改名送出後，會直接刷新同一個設定面板，方便連續操作。

## 建議權限
- 設定類指令：`Manage Server`
- 刪除訊息：`Manage Messages`
- 音樂：`Connect`、`Speak`
- 私人包廂：`Manage Channels`、`Move Members`
